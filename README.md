# Strata founder funding prototype

Interactive, UI-only prototype for growth-stage founders raising $50Kâ€“$20M. Built with Next.js (App Router), TypeScript, and TailwindCSS. All data is static from `/src/mock/*.json`.

## Getting started
```
npm install
npm run dev
```
App serves on `http://localhost:3000`.

## Routes
- `/` landing page
- `/how-it-works` process overview
- `/apply` intake wizard (UI only)
- `/dashboard` readiness + pipeline overview
- `/company` company profile
- `/funding-options` route selection
- `/roadmap` execution roadmap
- `/investors` investor discovery + intro requests (UI stateful)
- `/deal-room` mock document manager (UI stateful)
- `/status` readiness snapshot
- `/messages` notifications feed
